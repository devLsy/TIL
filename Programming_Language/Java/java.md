# 람다 표현식(Lambda Expression)
* 람다식을 사용하기 위해서는 구현할 인터페이스(반드시 함수적 인터페이스만 가능)가 필요 -> @FunctionallInterface 어노테이션으로 명시 가능 <br>
* @FunctionallInterface가 적용된 인터페이스는 한개의 추상 메소드만 선언할 수 있음, 메서드 추가 시 컴파일 에러 발생 <br>
