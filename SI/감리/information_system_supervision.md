# 정보시스템 감리
## 감리관련 
* 최초 감리는 계획대로 가고 있는지가 key point(요구사항 정의서, 요구사항추적매트릭스 등 체크) <br>
* 중간 감리는 설계가 제대로 되어 있는지가 key point(erd, 테이블 명세서 등을 체크) <br>
* 최종 감리는 화면을 보면서 success, fail을 o,x로 체크 <br>
